# RotaryEncoder
This is small and fast Arduino library for Rotary Encoder with interrupts. **Debounce capacitors to ground required!!! Otherwise, skipped steps and unstable performance may occur.**

Supports:
- Arduino AVR
- Arduino ESP8266
- Arduino ESP32
- Arduino STM32
